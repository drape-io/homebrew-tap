# Homebrew Tap for Drape

This is the official [Homebrew](https://brew.sh/) tap for [Drape](https://drape.io) tools.

## Installation

```bash
brew tap drape-io/tap
brew install drape
```

## Updating

```bash
brew update
brew upgrade drape
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `drape` | CLI for uploading test results and coverage to Drape |
